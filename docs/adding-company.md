# การเพิ่มบริษัท

1. เพิ่มโลโก้ใน `src/assets`
2. เพิ่มบริษัทใน `src/config/companies.ts`
3. กำหนดบัญชี Brand, Report logo และ Theme
4. เพิ่มบริษัทและรายชื่อเตาใน MariaDB
5. สร้าง API key แยกเฉพาะบริษัท
6. กำหนดแหล่ง MQTT/HTTP และ mapping หมายเลขเตา
7. ทดสอบ Tenant isolation ก่อนเปิดใช้งาน

Frontend ไม่เก็บรหัสผ่านและไม่มีข้อมูลทดแทน ข้อมูลบริษัท เตา ผู้ใช้ รอบอบ และสิทธิ์ต้องมาจาก API/MariaDB เท่านั้น

ห้ามตรวจบริษัทด้วยชื่อบัญชีแบบ `startsWith` หรือ hardcode ในหน้า UI การแยกบริษัทต้องตรวจซ้ำที่ API และฐานข้อมูลเสมอ
